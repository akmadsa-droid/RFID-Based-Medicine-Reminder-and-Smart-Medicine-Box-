Upon startup, the Arduino initializes all components the LCD, DS3231 RTC, MFRC522 RFID reader, servo motor, buzzer, and LEDs  and displays 'Smart Medicine System Ready' on the LCD for two seconds before entering the main loop. Inside the loop(), the system continuously reads the current time from the RTC and displays it on the LCD as long as no medication alert is currently active. The checkSchedule() function runs every 30 seconds and cycles through five time slots sequentially (ledStep 0 to 4). When a slot activates, the activate() function is called, which sets medicineActive to true, lights up the corresponding LED for that slot Red for slot 0, Green for slot 1, Blue for slot 2, Orange (mixing pins 5 and 6) for slot 3, and Purple (mixing pins 7 and 9) for slot 4 and displays 'MEDICINE TIME!' on the LCD. The buzzer begins sounding a 1000 Hz tone immediately.
 
The system then waits for an RFID card tap via checkRFID(). If the card's UID matches the authorized UID {0x01, 0x02, 0x03, 0x04}, the buzzer stops, the servo motor rotates to 90 degrees to unlock the medicine box, and the LCD sequentially displays 'Access Granted', 'Welcome Kurt!', and 'time for your Medicine!'. The medicineTaken flag is set to true and a 5-second countdown begins. After 5 seconds, finishCycle() is called the servo returns to 0 degrees (locked), and the system resets via resetSystem(), which clears all LEDs, resets all flags, and returns the display to showing the current time.
 
If an unrecognized card is tapped, the LCD displays 'Access Denied!' for 1.5 seconds and the box remains locked. If no card is tapped within 10 seconds of the alert starting (millis() - reminderStart > 10000), the missedDose() function is triggered the buzzer stops, the LCD shows 'You missed your Medication!' for 3 seconds, and the system resets to idle state.
